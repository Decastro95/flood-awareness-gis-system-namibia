# Flood Awareness GIS System Prototype

URL: https://flood-awareness-gis-system.vercel.app

## Description
Web-based GIS for public flood awareness in Northern Namibia. Visualizes flood-prone zones, safe areas.

## Key Features
- Interactive map with risk layers.
- Real-time weather from OpenWeatherMap.
- Safe zones with shelters.
- Alerts and seasonal indicators.

Emergency Contacts: General 112, Police 10111.
